## Nomenclatura
Ex.: 100GBase-SR4
- 100G - Velocidade
- Base -> Sinalização (atualmente todos os padrões ativos são base)
- S -> Meio físico
	- C: Copper
	- T: Par trançado
	- F: Fibra
	- S: Fibra short
	- L: Fibra L
	- E/Z: Fibra óptica
- R -> Codificação de bits
	- X: 8b/10b
	- R: 64b/66b
- 4 -> Quantidade de transmissões paralelas 
	- Mídias paralelas
	- [[WDM|WDM]]
